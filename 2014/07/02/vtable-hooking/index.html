VTable Hooking

Introduction

Today, I will try to explain how we can make a hook on a C++ class method. This technique working on Linux and Windows, but my example is compiled on Linux.

VTable, what is it?

C++ use the concept of inheritance. We will use two class in this article:

B have three methods:

  • This constructor: B()
  • The overloaded method Hello()
  • And the inherited method Useless()

And a m_value member.

To see the compiler problem to compile it, letâ€™s see an example:

At the compilation time, how the compiler can know that varB->Hello() is a reference to A::Hello() or B:Hello()? Because of it, when a C++ program is compiled, the compiled code will use a Virtual Call Table (VTable):

*A  ->  &VTable (ex: 0x401111)  ->  &Hello      (ex: 0x40AAAA)
        m_value (ex: 0x1337)        &Useless    (ex: 0x40BBBB)

*B  ->  &VTable (ex: 0x402222)  ->  &Hello      (ex: 0x40CCCC)
        m_value (ex: 0x1338)        &Useless    (ex: 0x40BBBB)

With this process, just with the object address we can call the right method!
